# ML_Rain_Prediction_AI
Machine Learning Model that analyses meteorologic data, and predicts rain for the following day
## ML Algorithms

* Linear Regression
* KNN
* Decision Trees
* Logistic Regression
* SVM

## Evaluations Models

* Accuracy Score
* Jaccard Index
* F1-Score
* LogLoss
* Mean Absolute Error
* Mean Squared Error
* R2-Score
